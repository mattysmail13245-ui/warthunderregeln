§1 Aufnahmepflicht (Stream) 🎥
Sobald du dich im Discord befindest und War Thunder spielst, muss dein Stream aktiv sein. Eine Ausnahme gilt ausschließlich dann, wenn du eine offizielle Genehmigung erhalten hast, deinen Stream pausieren zu dürfen.
// Genehmigungen sind vorab bei der Serverleitung einzuholen //
§2 Höchstes Gut 🏆
Peniße sind das höchste Gut.
Dieser Grundsatz ist selbsterklärend und bedarf keiner weiteren Erläuterung. Er wird ohne Ausnahme anerkannt.
// Kernwert des Servers — unveränderlich //
§3 Markierungspflicht 🎯
Sobald ein Gegner gesichtet wird, besteht die sofortige Pflicht, diesen zu markieren. Das Unterlassen gilt als Verstoß gegen die Teamkoordination.
// Scouting ist Teamarbeit — jede Sichtung zählt //
§4 Lufthoheit ✈️
Es darf sich zu jedem Zeitpunkt nur eine einzige Person in der Luft befinden. Mehrfachbesetzung des Luftraums ist untersagt. Absprachen im Voice-Channel sind vor dem Aufsteigen verpflichtend.
// Wer zuerst in der Luft ist, hat Vorrang //
§5 Keine Grauzonennutzung ⚖️
Das bewusste Ausnutzen von Grauzonen, Regelungslücken oder unklaren Formulierungen ist strengstens untersagt. Der Geist einer Regel gilt ebenso wie ihr Wortlaut. Die Serverleitung entscheidet im Zweifelsfall abschließend über die Auslegung.
// Unwissenheit schützt vor Strafe nicht — im Zweifel nachfragen //
⚠ Sanktion: Verwarnung bis permanenter Ausschluss
SONDERREGELUNG
§6 Verbotener Begriff — Herz Küssen 🚫
Die Verwendung des Begriffs „Ich küss dein Herz" sowie aller sinngemäß gleichwertigen Formulierungen ist auf diesem Server strengstens untersagt.

Bei Zuwiderhandlung erfolgt ein sofortiger und permanenter Ausschluss vom Server — ohne Vorwarnung und ohne Einspruchsmöglichkeit.
